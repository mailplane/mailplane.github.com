Yes, check if "Keyboard Shortcut" is enabled.

  1. Open you Gmail settings page by clicking the "Settings" link in the upper right part of the main window.
  2. Set the "Keyboard shortcut" preference to ON.
  3. Save settings.
  4. Click the View > Reload Page menu item.
