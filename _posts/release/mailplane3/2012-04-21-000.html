This version is still based on the same WebKit and thus doesn't solve the stability issues, yet. After running into a number of dead-ends we're seeing the lights at the end of the tunnel which brings us to a more stable version of Mailplane 3.

Download animation
Animation which indicates the start of a download.

Open Web Links in Background
Opens web browser in background when pressing the "Option" key while clicking an external link.

Fixes
* Google Bar is now handled the same way as in Gmail
* Attachments do now work with replies as well
