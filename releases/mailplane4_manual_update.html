Mailplane 4 - Manual Update

  1. Select 'Mailplane 4 > Uninstall' from the menu to delete your settings.
  2. Move Mailplane 4 in Applications to the Trash
  3. Get the lastest version with this Download link
  4. Install Mailplane. It will import accounts and settings from Mailplane 3 on startup.
