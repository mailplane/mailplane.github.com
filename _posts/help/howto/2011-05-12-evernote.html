Remember and track your email conversations with Evernote.

With the new Mailplane Evernote integration you can easily create notes linking to a Mailplane conversation and import attachments to Evernote.

Creating Notes

To create a note, open a conversation in the main window and either click on:

  • The 'Evernote' toolbar item (Tip: click the 'View > Customize Toolbar' menu item to add the toolbar item to your toolbar) OR
  • Use the Edit > Create Evernote Note menu item

If you select some text, the selected text is copied to the note. If no text is selected, the text of the last message is used as the note text.

Toolbar Icon: To add the Evernote icon to your toolbar, follow these steps:

  1. Click View > Customize Toolbar menu item
  2. Drag the green Evernote icon to the toolbar
  3. Click the "Done" button.

Evernote will open right after you create the note:

You can review your note and you have the opportunity to:

  • change the title & text
  • add tags to your note
  • remove the note using the 'Delete' toolbar item

From Evernote, you can open the linked conversation in Mailplane by clicking the note link. This is handy, if you'd like to reply to the person who wrote the message or to read the whole conversation.

Import Attachments to Evernote

Here's how you can import an attachment

1. Download the attachment

2. Click "Import" in the download window

Tip: You can also use the Download > Import 'filename.jpg' to Evernote menu item.

3. Review the note: add tags, change title.

Tip: If you'd like to import a whole photo collection, download the ZIP file. On importing, Mailplane will unpack the ZIP file and create a note containing all the pictures.

Requirements

To use the Evernote integration, you need to register for a free Evernote account and download and install the Evernote for Mac application.

Using keyboard shortcuts

By default, the Evernote menu items do not have keyboard shortcuts. If you'd like to assign a keyboard shortcut, please watch this video for instructions.
