See how you can setup to use Mailplane with offline Gmail. Includes Google Gears installation.

Requirement: Mailplane 2.1-beta: Visit the mailplaneapp Google Group to get the latest build. Watch for the "Preview Builds" thread, the last post contains a link to the lastet build.

If you have already installed Gears and you like to redo the installation:

  • To delete local data store: Open Finder, go to /Users/USERNAME/Library/Application Support/Google and delete "Google Gears for Safari" (do a backup if needed).
  • To completely Uninstall Google Gears check out the Google documentation.
