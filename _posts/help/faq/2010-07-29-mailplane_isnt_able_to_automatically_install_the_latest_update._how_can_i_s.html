In some rare cases, the auto-updater isn't able to install the latest release. Here's how you can update Mailplane manually:

  1. Close Mailplane
  2. In Finder: Open the Applications folder and move the Mailplane application to the Trash
  3. Open the Mailplane download page in your web browser.
  4. Click the download link for your Mac OS X version (see Apple > About this Mac menu item for the version number).
  5. Open the DMG file
  6. Drag the Mailplane icon to your Application folder
  7. Start Mailplane.
