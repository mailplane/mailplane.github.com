New Compose Mode - Second Take
Mailplane supports now Gmail's inline window and doesn't force composing a message in a new window anymore. You can still compose a message in a separate window by using shift while clicking the compose button or doing Command-N.

Fixes
* Print dialog doesn't stick behind the QuickLook panel anymore
* 'Show Original' doesn't download as text file anymore
