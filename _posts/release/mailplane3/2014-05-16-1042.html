Disable Insert Images Inline
We added a preference to enable/disable the 'Insert images inline' functionality. Simply go to Preferences... > Attachments and uncheck 'Insert images inline'.

Improvements
* Cursor doesn't disappear anymore while composing a message
* Automatically reduces inline images if they are wider than 500px
* Doesn't crash anymore if opening a document which has no default app
