A lists of Mac applications, which integrate with Mailplane.

  • BusyCal is an iCal replacement with full Mailplane support.
  • Paperless supports Mailplane to send emails with attachments. Check out this thread for more info.
  • Use the OmniFocus/Mailplane clipping plugin to easily create tasks linking back to the email conversation. If you like to share your OmniFocus projects with other people or access your OmniFocus in your browser, check out www.spootnik.net.
  • Use iWork (Numbers, Pages, Keynote) "Share" menu to send email with iWork, Office documents, or PDFs as attachments.
  • Things, a very nice GTD client, supports Mailplane out of the box. Create new tasks from Mailplane.
  • Use Mailplane's iPhoto plugin to email pictures. Check out Mailplane > Preferences > Photos > iPhoto plugin
  • GrandTotal, an invoicing application for designers, lets you email invoices with Mailplane.
  • Google's Picasa automatically composes emails via Mailplane when Mailplane is the default email client.
  • Send Word documents from MS Word, EXCEL, Powerpoint.
  • Use Mailplane Service menu from Finder
  • Screensteps is great to create how-to documents. And it integrates with Mailplane.
  • Integrate any other application using Mailplane's AppleScript interface.
  • Use it with Application Launchers like Alfred, LaunchBar, or Quicksilver.
  • Use Tags to quickly find related emails and files.
  • Mailplane's URL drag and drop works great with Alarmsapp
  • Startupizer is a login items manager. It supports Mailplane and even uses the Mailplane icon inside the Startupizer icon.
  • Lyn, a fast and lightweight image browser, lets you directly email pictures. More information

Is there an application missing? Please let me know and it'll be added.
