Embedded Images
Provides similar download functionality for embedded images as Mailplane already does for normal attachments.

Hide Ads in Promotions Tab
"Preferences... > Style > Hide Ads" does also hide ads in Gmails Promotions Tab.

Hide Right-Hand Column
"Hide Ads" (Preferences... > Style) just hides Ads. Disabling "Hide Right-Hand Column" will still hide Ads but the right-hand column (People information, Streak's Boxes, etc.) will remain visible.

Fixes and Improvements

  • Don't ask 2-step verification code again if "Trust this computer" has been selected
  • "Download As..." doesn't change the default download folder anymore
  • Streak 6.7
