Automatic Sign-Out

Leave the password empty in 'Preferences... > Accounts > General' and cookies and caches get cleared automatically when Mailplane quits.

Calendar Reminders

Go to Google Calendar lab settings and enable 'Gentle Calendar Reminders'. That way you'll get Calendar reminders as OS X notifications. More...

Hot Keys

Define hotkeys in 'Preferences... > Shortcuts' to compose a message, compose a note to self or toggle 'Do not disturb'.

Print Gmail Header

Hide the Gmail header when printing an email via the 'Show Gmail Header' setting in the print dialog.

Multiple Account Stability

And last but not least we moved away from Google's multiple signin functionality in June this year which does also remove Google's previous limitations. It was a big under-the-hood improvement.

We still suggest using our 'Switch to Account...' functionality if you have a lot of accounts to manage but technically speaking, your Mac's memory and screen estate is the only limit.
