In most cases the crash is caused by a corrupt keychain. Here is how you can work around the problem:

  1. Start Mailplane while holding the Option key: This will shows the Authentication dialog box
  2. Disable the Automatically login when needed option
  3. Disable the Store password in keychain option
  4. Login

Now that Mailplane works, you may want to repair the keychain. Please check this Apple page.

If Mailplane is still crashing, please report your crash to us.
