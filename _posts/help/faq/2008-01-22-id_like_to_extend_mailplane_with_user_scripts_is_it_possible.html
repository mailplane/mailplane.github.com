Yes, it is. Thanks to the excellent work of KATO Kazuyoshi there is an extension called GreaseKit. GreaseKit allows you to install user scripts in any WebKit application like Mailplane or Safari. This blog post shows you how to install and run a sample script called Folders4Gmail user script. Important: This is an example only, as the 'Nested Labels' Gmail lab feature supersedes this script.

Tip: You need to activate the script for Mailplane so it can do its magic:

  1. Open Mailplane
  2. Go to the GreaseKit menu
  3. Select the Folders4Gmail menu item, it must have a checkmark as a result
  4. Restart Mailplane to apply changes.
