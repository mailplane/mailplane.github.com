Insert Images Inline
Mailplane supports inline images. You can drag and drop images, paste them or insert them via our 'Mail > Insert' menu.

Edit Attachments and Reply
Right-click on an attachment reveals an 'Edit' context menu item which opens the document. Thereafter you'll find an 'Attach' menu item which composes a reply and attaches the edited document.

Inbox Type Detection
Mailplane does now automatically detect the right inbox type setting for your notifier. It makes sure that the tab count and notifier is always in sync.

Custom Notification Sounds
Custom sounds are now supported. Learn more.

Improved Spinner Look
A big thank you to Steve from Irradiated Software for sharing his spinner implementation. Mailplane's load indicator (inside the tab) now has an indented look.

Improvements
* Image Upload in Hangout Chat doesn't crash Mailplane anymore
* Safari 'Email this page' doesn't truncate URLs anymore
* Support for OmniFocus' AppStore edition
* Opens links in Calendar externally by default
