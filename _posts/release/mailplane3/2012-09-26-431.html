Recent Gmail Changes
Google is rolling out a new version that breaks Mailplane. This update is able to handle both versions.

Event Invitation
Event Invitations open now inside the Calendar tab instead in a new window.

Attachment Download
Some users experienced crashes while starting a download. This should be resolved now. And we made the magnifier easier to reach if there's scrollbar.

Keyboard Shortcuts
* Changed "Toggle Selection" shortcut to Command-D
* Changed "Open In New Window" to Command-[ while being in detail or compose mode
