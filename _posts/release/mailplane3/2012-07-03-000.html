Gmail Chat
There's a new Account preference which allows to unblock Gmail Chat. Unfortunately the stability and performance issues still remain if using Gmail Chat in multiple account simultaneously.

Keyboard Shortcuts for Google Calendar
Mailplane supports now iCal like Keyboard Shortcuts while being inside Google Calendar.

Private Cookie Store
Mailplane uses a private cookie store again. Thus, you can use Safari with a different or no Google identity while being signed-in in Mailplane.
