Release notes:

  • NEW: Added Window > Enter Full Screen menu item and keyboard shortcut
  • CHANGED: Removed Mail > Forward keyboard shortcut because of Full Screen mode
  • FIX: "Your action was not successful" error message when sending messages
  • FIX: Attachments could not be uploaded to the server
  • FIX: Chat in left sidebar couldn't connect to the server
  • FIX: Two-finger swipes for back/forward didn't work (Lion only)

Update (2011-08-29): Mailplane 2.5.1 has been submitted to the Mac App Store.
