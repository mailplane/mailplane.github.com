Use text snippets to insert canned responses, greetings and more.

Create a new text snippet

  1. Go to "Preferences > Accounts" and click the "Text Snippets" tab
  2. Click the "plus" button to add a new snippet
  3. Double click the entry and enter the text snippet name. The name is used to identify the snippet and to search for it.
  4. Enter the text snippet

Insert a text snippet

  1. Use the "Mail/Insert/Text Snippet..." menu item (or Command-Option-I) to open the text snippet window
  2. Search for the snippet
  3. Use the up/down arrow keys to select the desired snippet and press ENTER to insert the text snippet into the email

Tip: Check out Mac OS "Text Substitutions"
