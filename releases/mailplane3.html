Recent Mailplane 3 Releases

April 10, 2013 (#646)

TaskForce & RightInbox
TaskForce plug-in is back in Mailplane 3 and we added RightInbox. RightInbox was kind enough to give Mailplane users a 30% discount until May 15th. We urge you to check them out!

Compose Message With...
Compose messages with different accounts directly from the Dock. Close the main window (Command-W) and you're able to send email without having to look at email. Thanks to John and Merlin Mann for the idea!

Reset Mailplane
Some of you might already know it. "Reset Mailplane" is our secret weapon :) Next to removing Cookies, it does now also clear Caches, close all windows and tabs and restarts Mailplane.

LetsMove Integration
LetsMove is a tiny but very helpful addition. If it's not already in the Application folder, Mailplane ask you now on start-up if it should move itself to the Applications folder.

'Add Cc Recipients' Shortcut
Gmail's new 'Add Cc recipients' Shortcut happened to conflict with 'Format > Font > Show Colors'. Thus, we changed Show Colors to Command-Option-C. Thanks to Daniel for the idea!

Fixes
* Viewing Excel sheets in QuickLook caused connection problems
* Toolbar buttons look now crisp on Retina screens
* Attempt to make Notifier more responsive and fixed a crash
* Gmail doesn't steal focus anymore

March 16, 2013 (#622)

Image Resize Dialog
Mailplane 3 doesn't show the resize dialog anymore if the image is below a minimum file size. Go to Mailplane 3 > Preferences... > Attachments to define it.

Copy/Paste to Attach Files
Open Finder, select some files, copy them to the clipboard and paste them into a new message inside Mailplane.

Print Dialog
The print dialog does now include page orientation.

Fixes
* FIX: Mailplane was holding back from sleep after downloading a file
* FIX: Crash while creating a message with an attachment
* FIX: Mailplane quits unexpectedly after opening an external link
* FIX: Crash while closing tabs on Mac OSX 10.7.5

February 15, 2013 (#598)

Alias in Download Location
Supports Aliases as download location (Preferences > General > Save attachments to).

Crash Reporter
Mailplane 3 has now a crash reporter which appears on start up after a crash. Don't hesitate to send us your crash reports.

Fixes
* Supports Evernote 5.0.5
* Sheets do not block the whole application anymore.

January 24, 2013 (#587)

Scale Images Dialog
Scale image dialog doesn't appear anymore if the image is smaller than the lowest option (1024 x 768 px).

Fixes
Attachments didn't work anymore within the new compose mode due to a recent Google change.
