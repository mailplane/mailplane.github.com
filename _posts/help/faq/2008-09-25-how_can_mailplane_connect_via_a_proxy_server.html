Here is how you get the proxy working:

  1. Enable the proxy in your network preferences
  2. Start Safari and login Gmail, authenticate the proxy server if needed
  3. Switch from http to https in the address bar and make sure you can access Gmail, too

Try to use Mailplane thru your proxy server. If it doesn't work, please try these steps:

  1. Open keychain access application and search for the proxy login items
  2. Give Mailplane access to these items
  3. Start Mailplane
