New Gmail UI
Supports the new Gmail UI which Google introduced recently. It allows you to organize emails directly from your inbox, snooze emails until later and use suggested replies.

Create Event in separate window
Similar to composing a message in Gmail, you can now create an event in a separate window via 'Calendar > New Separate' (Command-Shift-N).

Fixes and Improvements
  • 'Edit > Save to Todoist' adds a back-link to the Todoist title.
  • Updated to Streak 6.32.
