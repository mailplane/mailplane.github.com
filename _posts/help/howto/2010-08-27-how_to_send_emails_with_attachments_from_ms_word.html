Use an Automator Workflow to send emails from MS-Word (Office 2008) with Mailplane.

Send the current Word document as PDF

  1. Open a document inside Microsoft Word
  2. Select the File->Print menu item
  3. Click the PDF button
  4. Choose Mail PDF with Mailplane

Or: Send the current Word document

Installation:

  1. Download this Automator workflow
  2. Uncompress the downloaded if needed
  3. Place this workflow into the 
    Home:Documents:Microsoft User Data:Word Script Menu Items

Usage:

  1. Open Microsoft Word
  2. Open a document or create a new document
  3. Important: Make sure all changes to the document have been saved
  4. Click the script menu (it's right of the Help menu)
  5. Select the Send Document with Mailplane menu item
