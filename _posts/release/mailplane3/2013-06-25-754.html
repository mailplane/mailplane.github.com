Hide Tab Bar
"View > Hide Tab Bar" allows to hide the tab bar if only one tab is open.

Keyboard Shortcuts - 2nd Take
Gmail shortcuts work now in reply mode as well. It required us to rename "Mail > Open" to "Mail > Open Conversation" and to introduce a new "Mail > Open In New Window" menu item.

Fixes
* AppleScript: Supports paragraphs in mail body
* Download attachments containing special characters in file name
* iPhoto (Media > Photos) works now in file dialog
* Opening a file dialog doesn't crash in Lion (10.7) anymore
