Make image attachment smaller and convert them to a format everybody can open.

Todays digital cameras produce very big photos, suitable for creating high quality prints. When you send them by Email on the other hand, they should be reduced in size and converted to JPEG.

Normally when you attach one or pictures (using drag and drop or using the clipboard), you'll get this dialog to optimize your pictures:

  1. Optimize picture attachments controls all other settings on this page. If turned off, the picture will be attached "as-is" and will not be modified in any way.
  2. This preview shows the pictures which are attached.
  3. Choose the picture size (Large, Medium, or Small)
  4. Select the desired quality setting, less quality means smaller size.
  5. Check Don't show this dialog again if you're always using the same settings. You can always turn it back on in Preferences > Photos.
  6. Click "Attach" button to attach the pictures.

Tip: Command-Drop turns off optimization

If you like to send files "as-is" without any change, but you don't want to change the default settings, you can hold down the Command key while dropping the file. No dialog will be shown as a result.
