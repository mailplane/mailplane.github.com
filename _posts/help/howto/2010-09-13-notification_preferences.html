Mailplane can use different ways to tell you about new and unread messages in your inbox.

To configure Mailplane's unread message notifications, please open "Preferences > General".

  1. Enable "Display status item in the menu bar" to show a little letter icon in the menu bar. The icon turns red when you have new mail. It also shows the number of unread messages. Click the icon to switch Gmail accounts and to directly access the most recent unread messages.
  2. Choose your favourite "new mail" sound. You can also choose a custom sound to be played. Tip: Each Gmail account can play its own sound, see below.
  3. Choose when the sound should be played:
    • "No sound:" Never play a "new mail" sound.
    • "Active account:" Only play the sound when new mail arrives for the current activate Gmail account.
    • "All signed-in accounts:" Play sound for all signed-in accounts. Use this setting when you want to monitor "background" accounts, too.
  4. The unread message counter is displayed in the application icon and next to the menu bar item. Choose how the unread message counter is calculated:
    • "No account:" Do not show an unread message counter.
    • "Active account:" Only display the unread message counter for the active account.
    • "All signed-in accounts:" Add all unread message counters from the signed-in accounts. Use this settings if you like to monitor "background" accounts, too.

  5. Mailplane integrates with "Growl" to display new mail notifications. To install "Growl" visit this website.

    • "No account:" Do not show any "Growl" notifications
    • "Active account:" Display Growl notifications for the active account only
    • "All signed-in accounts:" Display new Growl notifications for all signed-in accounts. Use this settings if you like to monitor "background" accounts, too.

Account notification settings

You can further customize Mailplane's new mail notifications for each Gmail account. Please click the "Accounts" tab in the preference window:

  1. Assign a nickname to your account. The nickname is displayed in the main window title and in the growl notification if the "All signed-in accounts" setting is enabled for "Growl".
  2. Choose a color for your Growl notifications. The color is only displayed when the "All signed-in accounts" setting was choosen.
  3. Sign-in in the account when Mailplane is started. Useful for the "All signed-in accounts" setting. You can always manually sign-in/sign out accounts by using the account drawer (View > Show Account Drawer menu item) or menu bar item.
  4. Choose which label is used to calculate the unread message counter:
    • "Inbox:" This is the default setting
    • "Priority Inbox:" Uses the "Important" message counter provided by "Priority Inbox"
    • "Custom:" Choose any label to calculate the message counter. Just make sure the label is visible in the left-hand Gmail sidebar.
  5. Choose the custom label by pressing the "F5" key.

Tips and Tricks

  1. Use the "Do not disturb" feature to turn off all notifications.
  2. Use iCal to set the "Do not disturb" mode.
